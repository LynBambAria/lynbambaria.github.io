markdown学习

耗时两天遇到各种各样的问题，终于搭建好了个人博客，遇到的第一个问题就是如何写博客，此文章便应运而生。



markdown学习

author：凛竹九歌（LynBambAria）

一、标题

#一级标题
##二级标题
###三级标题
……
######六级标题（最多支持六个）

二、引用

使用方法 > +内容
如果在第二行 >> 就会嵌套

这就叫引用

甚至可以二层嵌套

那应该也可以三层

三、列表

  1. 无序列表
  • 列表1（- 列表1）
    • 列表2（+ 列表2）
    • 列表2的分身
      • 列表3（* 列表3）

说明-、+、*（空格）都可以用来表示列表
偶然发现如果列表如果没有对其不在同一行会在实心和空心之间变换欸

  1. 有序列表
    数字+.+空格+内容

  2. 嵌套

    1. 这
    2. 就
    3. 是
    4. 嵌
    5. 套

      值得一提的是如果要实现嵌套需要（痛击空格4次）+数字+.+（空格）+内容

  3. TodoList(我觉得应该叫勾选框是不是更合适)
    使用方法- [ ] 输入选项（[ ]一定要有一个空格，否则就变成列表了

    • a
    • b
    • c

四、表格

左对齐 居中对齐 右对齐
a b c 
| 左对齐 | 居中对齐 | 右对齐 |    
| :---- | :----: | ----: |    
| a | b | c |

五、段落

  • 换行：此行结束两个以上的空格后回车/空一行
  • 分割线：三个（及其以上）个***……
  • 字体
    字体 代码
    斜体 * 内容 *
    ==高亮== == 内容 ==
    粗体 ** 粗体 **
    斜粗体 * *
    删除 ~~ 内容 ~~
    下划线 <u></u>
  • 脚注（我觉得叫引用是不是更好一点）
    你看看这是不是和引用参考文献差不多[^1]
    [^1]:使用方法就是在引用的地方[^x], 然后再后面单独一行再
    [^x]:输入引用内容即可

六、代码

这叫代码段，使用方法：在代码开头三个英文一撇，结束位置再加三个
printf("hello world");
有了代码段肯定也有代码行，与代码段类似，但是区别是只需要一撇

七、超链接

[其实直接输入链接就行了]：www.baidu.com
也可以这样我的博客
这个使用方法是[链接名称]（网站）

八、图片

图片的插入需要图床，免费图床链接：https://imgse.com
上传图片后下面可以生成markdown地址，可以直接贴过来

值得一提的是markdown只能直接贴图片，而html可以修改图片，比如图片位置，长宽等

九、插入数学公式

暂时用不到，先搁置一边罢（说了可以插入latex，不知道mathtpye行不行，有空研究一下）

十、html/css语法

等同于html和css语法，看来空闲时间也可以学习一下了
最明显的用途就是美化博客界面

十一、导出为PDF

右键preview，open in brower,右键打印为PDF
